# Twitter (X.com) UI Clone

A responsive Twitter (X.com) user interface clone built using **HTML**, **Tailwind CSS**, and **Vite**. This project recreates the modern X.com homepage with a clean, responsive design and pixel-perfect layout.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/dhanrajk046/Twitter-X.com-UI-Clone.git
```

Navigate to the project folder

```bash
cd Twitter-X.com-UI-Clone
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
